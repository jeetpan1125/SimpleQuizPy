# Simple Quiz Game
A category-based interactive quiz application written in Python.
The program allows you to choose a category and answer randomized questions with built-in error handling for invalid inputs.
This is my 1st Python project for GitHub.

## Features
- Three Categories: Geography, Math, and Computer Science
- Input Validation: Uses try-except to prevent crashes from typos or non-number inputs
- Randomized Questions: Shuffles the question order for every new game
- User Feedback: Provides the correct answer immediately if you guess wrong
- Retry Logic: Stays on the same question if an invalid number (like 99) is entered

## How to Run
1. Open the project in your preferred IDE
2. Run SimpleQuiz.py
3. Follow the prompts in the terminal
